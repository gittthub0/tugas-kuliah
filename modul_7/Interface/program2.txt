class Animal:  Dalam Java, sebuah interface adalah kumpulan metode abstrak yang dapat diimplementasikan oleh kelas-kelas lain.
    public void animalSound(); - Metode ini harus diimplementasikan oleh kelas lain
    public void sleep(); - Metode ini juga harus diimplementasikan kelas turunannnya

class pig: mengimplementasikan method yang ada di class Animal

class Main:  membuat object myPig dari class pig
    Memanggil method animalSound() dari object myPig
    Memanggil method sleep() pada objek myPig
